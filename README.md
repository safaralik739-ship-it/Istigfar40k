# Istighfar 40K

Android-приложение для подсчёта истигфара (أستغفر الله) с целью 40 000 (или любой другой).
Полностью офлайн, без рекламы и без регистрации.

## Технологии

- **Kotlin** + **Jetpack Compose** (100% UI на Compose, без единого XML-layout)
- **Material Design 3** — тёмно-зелёная/изумрудная палитра с золотыми акцентами
- **MVVM** + **Repository Pattern** (Clean Architecture)
- **Room** — история всех действий (+1, добавления, сбросы)
- **DataStore (Preferences)** — текущий счёт, цель, настройки, достижения
- **Navigation Compose** — нижняя навигация: Главная / Статистика / Достижения / Настройки
- **AlarmManager** — ежедневные напоминания (1–3 раза в день)
- **JUnit** — unit-тесты для основной логики счётчика (`CounterMathTest`)

## Как открыть проект

1. Установите **Android Studio** (Koala/Ladybug или новее).
2. Откройте эту папку через `File → Open...` — Android Studio сам подхватит `settings.gradle.kts`.
3. Дождитесь синхронизации Gradle (потребуется интернет для скачивания зависимостей при первом запуске).
4. Нажмите **Run ▶** на эмуляторе или физическом устройстве (Android 8.0 / API 26+).

Никаких дополнительных ручных настроек не требуется — проект собирается «из коробки».

│   │   ├── history/       # Статистика (сегодня/вчера/неделя/месяц/всего + график)
│   │   ├── achievements/  # Достижения (1К/5К/10К/40К/100К)

## Основные функции

- Большой счётчик `X / 40 000` с золотым кольцом прогресса и процентом выполнения
- Кнопка **+1** с вибрацией и анимацией нажатия
- Кнопка **−1** для отмены случайного нажатия (не уходит ниже нуля)
- **«Добавить количество»** — ручной ввод числа для тех, кто считает на механическом тасбихе
  (пример из ТЗ: ввели 350 → нажали +1 двадцать раз → итог 370, покрыто тестом)
- Выбор цели: 1 000 / 5 000 / 10 000 / 40 000 / 100 000 или своя цель
- Автосохранение после каждого изменения (DataStore + Room)
- Статистика: сегодня, вчера, неделя, месяц, всего, среднее в день, график за 7 дней
- История всех действий с датой и временем
- Достижения с конфетти при прохождении каждой вехи
- Настройки: вибрация, звук, напоминания (0–3 раза в день), тема (светлая/тёмная/системная),
  изменение цели, сброс счётчика, экспорт истории в CSV
- Полностью офлайн, без интернета и рекламы

## Тесты

Юнит-тесты чистой логики находятся в `app/src/test/java/com/istighfar/app40k/CounterMathTest.kt`.
Запуск: `./gradlew test` или через Android Studio (правый клик на файл → Run).

Покрыты: инкремент/декремент с ограничением снизу, ручное добавление количества (сценарий
с тасбихом 350 → +20 → 370), расчёт процента и остатка, обнаружение пройденных вех
достижений (включая одновременный проход нескольких вех), расчёт среднего в день.

## Примечание про иконку приложения

Иконка (`mipmap-anydpi-v26`) сделана как простой векторный placeholder (полумесяц + звезда на
тёмно-зелёном фоне с золотым акцентом) — замените `ic_launcher_foreground.xml` на свой дизайн
при необходимости.
